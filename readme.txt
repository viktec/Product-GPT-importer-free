== Description ==
Product GPT Importer Free ti permette di generare automaticamente prodotti WooCommerce partendo da file `.pdf`, `.docx` o URL.  
Usa GPT per creare titolo, descrizioni, attributi e immagini in modo intelligente.

== Installation ==
1. Vai in **Plugin → Aggiungi nuovo → Carica plugin**.
2. Carica `product-gpt-importer-free.zip`.
3. Attiva il plugin.

== Frequently Asked Questions ==

Sì, puoi impostarla nella pagina “Product GPT Importer” delle impostazioni.
